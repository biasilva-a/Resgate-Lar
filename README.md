🐾 Resgate & Lar: Website Institucional

 🌟 Sobre o Projeto

Este é o website institucional desenvolvido para a ONG "Resgate & Lar", focada no resgate, reabilitação e promoção da adoção responsável de animais.

O projeto foi construído com foco em **design responsivo**, garantindo uma experiência de usuário agradável tanto em desktops quanto em dispositivos móveis.

---

✨ Funcionalidades Principais

 Página Inicial (Home): Apresentação e destaques do projeto.
 Adoção: Galeria de animais disponíveis para adoção com modais informativos.
 Quem Somos: Detalhes sobre a missão, visão e pilares da ONG.
 Cadastro & Login: Páginas dedicadas com layout responsivo para autenticação de usuários.
  Design Consistente: Paleta de cores baseada em tons de bege e marrom (#F5EDE0, #A67B51, #d9985f) para um visual caloroso e acolhedor.

---

🛠️ Tecnologias Utilizadas
 HTML5: Estrutura semântica do projeto.
 CSS3: Estilização personalizada.
 Bootstrap 5: Framework CSS para componentes e responsividade do menu.
 JavaScript: Lógica básica de formulários e controle do menu collapse (Bootstrap Bundle).

---

⚙️ Como Executar o Projeto Localmente

Para visualizar e testar o website, siga os passos abaixo:

1.  Clone ou baixe o repositório: Obtenha todos os arquivos do projeto.
2.  Estrutura de Pastas: Certifique-se de que os arquivos `bootstrap.min.css` e `bootstrap.bundle.min.js` estão corretamente nas pastas `css/` e `js/` (respectivamente).
3.  Abra os arquivos: Basta abrir qualquer arquivo `indexX.html` no seu navegador (ex: `index4.html` para Adoção) para visualizar a página.

---

 🎨 Estrutura de Cores

A paleta de cores foi definida para transmitir aconchego e confiança:

| Variável | Código Hex | Uso Principal |
| :--- | :--- | :--- |
| Fundo Principal | `#F5EDE0` | Fundo das páginas/Lado "left" dos formulários |
| Destaque/Botões | `#d9985f` | Botões de Adoção/Links e títulos em destaque |
| Barra/Footer | `#DBC9B2` | Header e Rodapé |
| Fundo Escuro | `#A67B51` | Lado "right" dos formulários (Desktop) |
